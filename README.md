# DAT255-V26
Breast Cancer Histopathology Image Classification Using Convolutional Neural Networks: 
A comparison of Custom and Pretrained Models.


Using UV to create venv is recommended.


We have created a demo.ipynb notebook that demonstrates the performance of our best models by creating heatmap over mammography images of where cancer growth is detected.


The full dataset used can be found here: https://www.physionet.org/content/vindr-mammo/1.0.0/
Dataset is 53 GB compresed and over 300 GB unconpresed, therefore it is recomended to use the ZIP tools in the code to work on the compresed dataset.
